# global-development-hub

This civic-tech app automates how global development metrics from the UN and World Bank are collected, cleaned, and visualized. It is built to show off clean, optimised data pipelines and interoperable software architecture, making complex civic data easily accessible.

# 🛠️ Tech Stack
*   Data Pipeline: Python & Pandas
*   Backend API: Node.js & Express
*   Frontend Dashboard* ReactJS & Recharts

# 📌 Project Plan
-  Phase 1 (Data Pipeline): Build a Python script using Pandas to clean messy data and save it as a clean JSON file.
- Phase 2 (Backend API): Create a Node.js server that reads the JSON file and sends specific country data when requested.
- Phase 3 (Frontend Interface): Build a clean website template using React that fetches the data and displays it on a timeline chart.
